# ダッシュボード

cc-company の組織状況をブラウザで確認できるオプションツールです。

## インストール・起動

`.company/` があるプロジェクトで以下を実行します。

```bash
npx cc-company-dashboard
```

ブラウザが自動で開き、`http://localhost:3939` でダッシュボードが表示されます。

### オプション

```bash
npx cc-company-dashboard --port 4000    # ポート変更
npx cc-company-dashboard --no-open      # ブラウザ自動起動を無効化
npx cc-company-dashboard --dir /path    # .company/ を探すディレクトリを指定
```

## 機能

### Dashboard

組織の概要をひと目で確認できます。

- **TODO**: 今日の未完了 / 完了数
- **Inbox**: 未整理のメモ件数
- **Departments**: 稼働中の部署数
- **Today's TODO**: 優先度別のタスク一覧
- **Recent Activity**: ファイルの最近の更新履歴
- **Department Cards**: 各部署のステータスサマリー

### Explorer

`.company/` の全体構造をツリー表示します。

- フォルダの開閉（クリック）
- サブフォルダの階層構造
- ファイルをクリックすると部署詳細に遷移
- ステータスバッジ表示

### Graph

Obsidian 風のネットワーク可視化です。

- `.company` を中心に部署・サブフォルダがノードで表示
- ドラッグでノードを自由に配置
- スクロールでズームイン/アウト
- 背景ドラッグでカメラ移動
- ライト/ダークモード対応

### Search

全ファイルからキーワード検索します。

- リアルタイム検索（入力中に結果表示）
- ファイル名とコンテンツの両方を検索
- マッチ行をハイライト表示
- 結果クリックで部署詳細に遷移

### 部署詳細

サイドバーの部署名をクリックすると表示されます。

- 左: サブフォルダごとのファイル一覧（開閉可能）
- 右: Markdown プレビュー / 生テキスト切替
- frontmatter はプレビュー時に自動除去

## リアルタイム更新

ダッシュボードは `.company/` フォルダを監視しています。`/company` で秘書と作業した内容はダッシュボードに自動反映されます。

## ライト/ダークモード

トップバーの ☀/☽ ボタンで切り替えできます。設定はブラウザに保存されます。

## ダッシュボードなしでも動く

ダッシュボードはオプションです。`/company` の秘書機能はダッシュボードなしで完全に動作します。
